## Getting Started

##### Set up
Add Toucaan (the current repository) to your project:

```bash
    $ git submodule add https://github.com/Toucaan/toucaan
```

The command above will add Toucaan like a `git submodule.` That's it. You are all set to start using Toucaan to design your app's multiple interfaces now. ðŸ¥³
